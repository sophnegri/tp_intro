# Trabajo Práctico Bash

## Descripción
Script en Bash que permite gestionar archivos mediante un menú interactivo.

## Funcionalidades
1. Crear entorno de trabajo
2. Procesar archivos en background
3. Listar alumnos ordenados por padrón
4. Mostrar top 10 notas
5. Buscar alumno por padrón

## Estructura de carpetas
EPNro1/
- entrada
- salida
- procesado

## Requisitos
Definir variable de entorno antes de ejecutar:

export FILENAME=alumnos

## Ejecución
Dar permisos:
chmod +x menu.sh
chmod +x consolidar.sh

Ejecutar:
./menu.sh
